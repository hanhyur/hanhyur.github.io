1. 사용자가 상품 등록 페이지에 접근(HTTP GET /add)
2. 사용자가 상품 정보를 입력 후 서버로 전송(HTTP POST /add)
3. 상품이 성공적으로 등록된 후 Location 정보로 상품 정보 상세 경로를 Redirect로 응답
4. 클라이언트에선 응답 받은 정보에 있는 Location 정보로 Redirect하여 신규 상세 페이지로 이동

1. 사용자가 상품 등록 페이지에 접근(HTTP GET /add)

+ `TypeMissMatch`
+ 비즈니스 요구사항에 맞지 않음
